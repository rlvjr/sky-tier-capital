## Overview

Sky Tier Capital is a simulated enterprise IT infrastructure project designed to demonstrate practical experience with system administration, networking, and cybersecurity operations.

This project recreates the internal technology environment of a fictional financial trading organization, focusing on secure infrastructure deployment, identity management, documentation, and troubleshooting.

---

## Project Objectives

The goal of this project is to build and document an enterprise-style environment while developing hands-on experience with:

- Windows Server administration
- Active Directory implementation
- DNS configuration
- Group Policy management
- User and access management
- Infrastructure documentation
- IT troubleshooting practices

## Infrastructure Components

### Windows Server 2022

The foundation of the environment hosting core enterprise services.

Implemented services:

- Active Directory Domain Services
- DNS Server
- Group Policy Management
- Remote Server Administration Tools

---

### Active Directory

Provides centralized identity and access management.

Implemented for:

- User authentication
- Domain management
- Security groups
- Access control

---

### DNS

Provides internal name resolution required for domain functionality.

Used for:

- Domain communication
- Service discovery
- Infrastructure connectivity

---

### Group Policy

Provides centralized configuration management.

Used for:

- Security settings
- User restrictions
- System configuration
