## Overview

Group Policy Management was installed within the Sky Tier Capital Windows Server environment to provide centralized configuration management for domain-connected systems.

Group Policy allows administrators to apply settings and configurations to users and computers within an Active Directory domain.

## Purpose

Group Policy is used to manage and apply settings across an Active Directory environment.

Common uses include:

- Managing user and computer settings
- Applying security configurations
- Controlling system behavior
- Standardizing workstation configurations
